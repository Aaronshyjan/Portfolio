# Aaron Shyjan - Portfolio

Modern personal portfolio website built with Next.js 14, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS / ShadCN UI
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theme**: Dark Mode (Futuristic AI vibe)

## 📁 Sections
- Hero (AI Tagline & Intro)
- About Me & Education
- Technical & Soft Skills
- Professional Experience (Internships)
- Projects (IoT, ML, Android, Security)
- Research Work
- Certifications
- Contact & Social Links

## 🛠️ Usage
1. Clone the repo
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
